CSS

O CSS (Cascading Style Sheets) é uma linguagem de estilo utilizada para descrever a apresentação de um documento escrito em HTML ou XML. Ele funciona separando o conteúdo da apresentação, permitindo que você controle cores, fontes, espaçamentos, layouts e outros aspectos visuais de uma página web. O CSS pode ser aplicado de três formas: inline, interno (dentro de uma tag <style>) ou externo (através de um arquivo .css vinculado). Isso facilita a manutenção e a reutilização de estilos em várias páginas.
